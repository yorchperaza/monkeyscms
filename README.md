# MonkeysCMS Landing Page

Welcome to the MonkeysCMS Landing Page project.

## About the Framework

This project is built using **MonkeysLegion** as its core framework foundation.

- **Framework Skeleton**: [MonkeysCloud/MonkeysLegion-Skeleton](https://github.com/MonkeysCloud/MonkeysLegion-Skeleton)
- **Official Website**: [MonkeysLegion.com](https://monkeyslegion.com/)

MonkeysLegion provides the essential architecture and tools that power the MonkeysCMS ecosystem.

## Getting Started

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
